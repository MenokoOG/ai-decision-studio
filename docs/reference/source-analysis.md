# Source Analysis

This repository is based on four source artifacts:

1. **Strategic investment framework workbook**
   - Five-year cost and benefit modeling
   - One-time and annual costs
   - Risk mitigation cost lines
   - Running total and ROI-style rollups

2. **Cross-functional decision framework workbook**
   - Model, optimization, and infrastructure options
   - Cost, benefit, and risk comparison
   - Recommendation and governance framing

3. **End-to-end AI delivery roadmap deck**
   - Phase 0 to Phase 5 lifecycle
   - Functional lanes across leadership, data science, data, engineering, operations, and end users
   - Deliverables, milestones, test, go-live, and measurement concepts

4. **AI industry use cases workbook**
   - Use cases and template ideas to seed the app

## Product implication
The app is not just a calculator.
It is a decision-support and delivery-planning workspace for AI initiatives.

## Core modules implied by the sources
- Initiative workspace
- Business case engine
- Decision matrix
- Roadmap planner
- Template library
- Guarded AI assistant
- Import/export and audit trail
