# Product Brief

## Product name
**AI Decision Studio by Menoko OG**

## Repo name
`ai-cost-tool`

## Primary users
- CTO
- Product leaders
- Operations leaders

## Core promise
Help leaders evaluate, compare, govern, and plan AI initiatives with strong exports and optional guarded AI assistance.

## v1 priorities
1. Business case modeling
2. Decision comparison
3. Roadmap planning
4. High-quality export
5. Responsive premium UI
6. Optional AI provider connection

## Non-goals for v1
- Multi-user collaboration
- Cloud sync
- Auth system
- Full web search workflow
- Nest backend
